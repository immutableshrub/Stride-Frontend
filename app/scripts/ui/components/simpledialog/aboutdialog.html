Version 1.3.2

Stride is open-source software. Find the source on Github.

Stride is built with:

  • SVG.js - Canvas interaction and drawing support.
  • Vite - Bundler and build tools.
  • Mousetrap - Keyboard shortcuts handler.
  • UUID - Providing UUIDs.
  • Socket.io - Providing Websocket utilities.
  • Crypto.js - Providing encryption utilities.
  • Vercel - Hosting the frontend.
  • heroku - Hosting the backend.
